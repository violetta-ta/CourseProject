Installation guidelines:

To install and execute the code, you will need to ensure following Python libraries and packages installed:

- Python 3.8
- tensorflow 2.3.1
- tensorflow-hub 0.10.0
- scikit-learn 0.23.2
- pandas 1.1.4
- numpy 1.18.5,
if pandas is downloaded before this numpy will be downloaded already

Running the code:

Execute the python script as: “python dnn.py”, or run file "dnn.py" in your IDE.
